Welcome to mofh's documentation!
======================================

mofh is a package made for Python which makes the implementation of MyOwnFreeHost's API easier.

**Features**

- Easy to use.
- Synchronous and asynchronous wrapper.
- Covers all of the scopes provided by the API.
